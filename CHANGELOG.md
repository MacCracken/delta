# Changelog

All notable changes to Delta are documented in this file.

## [0.1.0] — 2026-03-10

### Added
- Project scaffold with Rust workspace (6 crates)
- `delta-core`: configuration, error types, repository and user models
- `delta-vcs`: bare git repository hosting via gix (init, delete, list)
- `delta-api`: axum HTTP server with health check endpoint (port 8070)
- `delta-ci`: workflow and pipeline data models
- `delta-registry`: artifact types and content-addressable storage interface
- `delta-web`: crate placeholder for web frontend
- TOML configuration with example config
- Development roadmap (9 phases)
- Architecture documentation
- Contributing guide
